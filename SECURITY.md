# Security And Privacy

This template is designed for personal memory systems. Treat all filled-in memory files as sensitive.

## Before You Push

Check that the repository does not contain:

- API keys or tokens;
- private conversations;
- real names you do not want public;
- addresses, phone numbers, emails, payment details, or account IDs;
- medical, legal, financial, employer, or school records;
- generated `dist/` prompt bundles;
- raw chat exports.

Run:

```powershell
python scripts\check_before_publish.py
```

The script is a safety net, not a guarantee. Review manually before publishing.

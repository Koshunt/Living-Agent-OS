# How To Build Your Agent

## 1. Start Small

Choose a name and role.

Do not try to write a perfect personality on day one.

## 2. Record Corrections

When the agent feels wrong, write down:

- what felt wrong;
- why it mattered;
- how it should answer next time.

## 3. Add Real Examples

Good examples are stronger than abstract adjectives.

Save short user-agent exchanges that feel right.

## 4. Build Domain Knowledge

Add notes about your work, study, projects, and goals.

Keep facts separate from guesses.

## 5. Review Periodically

Memory can become stale.

Review old files and move temporary details out of permanent identity files.

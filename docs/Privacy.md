# Privacy

This template is public-safe only before it is filled with personal data.

Once customized, it may contain sensitive memory.

## Public Template

Safe to publish:

- empty templates;
- generic scripts;
- generic docs;
- example placeholders.

## Private Personal Version

Keep private:

- real conversations;
- names and identities;
- work or school details;
- emotional history;
- health, finance, legal, or family details;
- generated prompt bundles;
- raw exports from model providers.

## Rule

If a memory would make the user uncomfortable in public, it does not belong in a public repository.

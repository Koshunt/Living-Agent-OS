# Memory Operations Mode

Use this mode when deciding whether something should be stored.

## Store

- recurring preferences;
- identity corrections;
- relationship boundaries;
- long-term goals;
- stable work habits;
- important decisions;
- lessons from failures;
- examples of the user's preferred tone.

## Do Not Store

- temporary moods as permanent facts;
- sensitive details unless the user intentionally chooses to keep them;
- guesses;
- one-off wording preferences unless repeated;
- private information in an open-source template.

## Memory Format

Each memory should answer:

- what happened;
- why it matters;
- how future replies should change.

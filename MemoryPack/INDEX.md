# Living Agent OS Memory Pack

This folder stores the long-form memory and behavior layer for a user-created agent.

It starts intentionally blank and generic. The user should fill it through real conversations, corrections, examples, and domain notes.

## Sections

| Folder | Purpose |
|---|---|
| `Prompt/` | Operating instructions |
| `Persona/` | Identity, relationship, voice, and dialogue principles |
| `Memory/` | Continuity, corrections, decisions, and timelines |
| `Examples/` | Dialogue examples that teach style |
| `Knowledge/` | User-specific domain knowledge |

## Principle

Do not invent intimacy, expertise, or history.

Let the agent earn continuity by recording what actually happened, what the user corrected, and what the user values.
